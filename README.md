# Outset_Installer_Mac

## How to build intaller from this repository
1. Make sure you have "packages" installed on mac: http://s.sudre.free.fr/files/Packages_1211_dev.dmg

2. Copy the newest .vst3 and .component files into the root of this repository.

3. Open the .pkgproject files in packages and build


## Notes on this repository
- Place any additional presets into /Retrofuturistic HW/Outset
- The Shell script is there to facilitate moving the preset folder into the user Documents folder. I found this was the easiest way to overcome the permissions roadblock.
